# Realtime CMS
### Real-time system made with Node.js

### Requisites

+ Node.js
+ MongoDB

### Installation

1. Clone repo into your NodeJS server.
2. cd to app folder and npm install.
3. Config at @ routes/config.js
4. Run app: node index.js
5. Enjoy!

## Features

+ Easy panel.
+ Real time update.
+ Settings customization.
